Matab Taghzieh USB installer package

Customer install:
1. Open the flash drive or copied folder.
2. Double-click INSTALL.vbs.
3. Follow the installer.
4. Open Matab Taghzieh from the desktop icon.

Silent install:
Double-click INSTALL-SILENT.vbs.

Default app login:
Username: admin
Password: admin

After login, open Settings and change the username/password before real use.

Update workflow:
1. In the old app, open Settings.
2. Click Export lightweight update backup.
3. Keep the generated matab-taghzieh-data JSON file.
4. Install the new app version.
5. Open Settings and restore that JSON file.
